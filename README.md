# bookmark for bootcamp23summer

## init

- voltaをinstall : https://docs.volta.sh/guide/getting-started
- `$ npm i -g @antfu/ni` (pnpm, yarn, npmなどのalias)
- `$ ni -g commitizen` (git czのためのライブラリ)
- `$ nci`
  - pnpmを選択

## dev

git commit
- `$ git cz` を使ってください

開発モード
- `$ nr docs:dev`

ビルド
- `$ nr docs:build`

プレビュー
- `$ nr docs:preview`
